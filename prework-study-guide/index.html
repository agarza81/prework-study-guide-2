Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML elements have an opening and closing tab
  • The head is the container for metadata
  • The body element is an example of a semantic HTML element

CSS

  • Inline, Internal and External style sheets are the 3 ways to style a web page using CSS
  • The CSS rule includes the selector, property/value, and the declaration
  • When creating and external CSS file, we need to link it to a HTML file
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git: connects to git code
  • git status: checks what branch we are currently on
  • git clone "link" : clones github repo to the directory
  • open cd name_of_repo: opens up the repo on computer
  • git add . OR git add -A : adds the repo to computer
  • git commit -m "unique name": gives the name of new
  • git push: connects to the git hub repo
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: resorts back to main branch
